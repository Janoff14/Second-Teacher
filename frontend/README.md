# Second Teacher — Frontend

Web UI for teachers, students, and admins will live here (e.g. Next.js or Vite + React).

For now the product API runs from **`../backend`**. Point the frontend `API` base URL at your deployed backend (e.g. `https://second-teacher-production.up.railway.app`) and set **`CORS_ORIGIN`** on the server to match the frontend origin.
